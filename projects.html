  • Trivollo

    Currently GratWise is working on the Android application called Trivollo. Trivollo is a trivia game where players test their knowledge in different areas. The initial mobile game will have thousands of questions and due to the game engine, a player will almost never see the same question twice. The game will contain plenty of achievements, competitions and other great features to keep everyone engaged. More detailed information will come soon...

    GratWise plans to release the first version of Trivollo before summer 2015.
